##NC News

NC News is a React-based news web application that interacts with a custom-built REST API. Users can browse articles, read comments, vote on articles, and post comments.

This project demonstrates frontend development skills including React components, state management, routing, and API integration.

**Live Demo**

https://ncnews2026.netlify.app

**Tech Stack**

React
JavaScript
Axios
React Router
CSS

**Features**

Browse a list of news articles
Filter articles by topic
Sort articles by date, votes, or comments
View individual articles
Vote on articles
Post comments
Delete user comments
Responsive layout

**Installation**

Clone the repository:

git clone https://github.com/jinsonpj/nc-news.git
cd nc-news

Install dependencies:

npm install

Start the development server:

npm run dev
Backend API

This project uses a custom REST API built with Node.js and PostgreSQL:

https://github.com/jinsonpj/backend-nc-news

**Minimum Requirements**

Node.js v18+
